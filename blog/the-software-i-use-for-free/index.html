大学里要用到的高端软件很多，但是作为一名穷学生自然都是买不起的。所以我们往往探索各种方法来尝试白嫖。然而找到一些安全又高质量的资源有时候并不容易，所以本文简单地汇总一下。

白嫖的各个等级定义

我把白嫖分为五个等级，安全性由低到高定义为：

  • Level1：安装网上下载的「破解版」。这种方法最不安全。很容易被植入病毒，特别是 Windows。一般情况下，强烈不建议用这种方法白嫖。
  • Level2：安装官方版后使用一些「启动器」之类，为独立于原软件的附加组件，需要在原软件运行时伴随运行。这种方法也有一定的危险性。任何不信任来源的程序长期在后台运行都是危险的。
  • Level3：安装官方版后使用「密钥生成器」之类的生成激活密钥，或者用某种「激活工具」修改某些文件达到激活目的。这种方法相对来说比较安全，因为该附加软件一次性运行，不保持在后台。当然也不是没有危害系统的可能。
  • Level4：安装官方版后通过某些开源工具或已知手段手动激活。如果 level2/3 中的工具是开源的（约等于可信任，但不全等于），或者不使用第三方程序而是通过检查源码等方式自己动手，则会更安全，几乎无后顾之忧。
  • Level5：安装官方版后通过官方渠道进行「学生认证」，获得免费使用权。优点是合法，缺点是大多数有一定有效期。

以及由于生态原因，一般 macOS 的安全性高于 Windows。我的 Windows 11 是装在虚拟机里，所以对于这方面的顾虑不是很大。

电子电路

HNU《电路与电子学》课程要用到的两个大型软件。当然，老师在学习通上提供了破解版……但是我并没有选择安装。原因一是来源不明，说不定是什么华军软件园搞来的，说不定还埋了挖矿病毒之类的。就像前文所述，我只信任官方版本。原因二是版本太老——没错，我就是有「最新版本」强迫症。最后最受不了的一点，他们居然把几个 G 的软件放在百度网盘上让我们下载……

NI Multisim

本软件白嫖等级为 level3。Windows 平台。

National Instruments 公司出品的 Multisim 是“行业标准”的电路仿真软件。

官网下载地址

国外有人制作了 NI License Activator 工具，点一点就能激活 NI 的软件。下载地址在这里。

Intel Quartus II

本软件白嫖等级为 level3。Windows 平台。

官网下载地址

⚠️ 修读 HNU《电路与电子学》课程的同学请注意：一定要下载 13.0sp1 版本，不能使用 13.1 版本！因为 HNU 工训中心要求我们使用的芯片是 Cyclone II 系列的，但是 13.1 版本开始不再支持这个系列。

网络上也流传着一个破解器，可以直接根据 Quartus II 生成的网卡 ID，生成一个密钥，替换一个 dll 文件后即可用密钥激活该软件。我上传到了我的服务器，下载地址是这个，其中有详尽的使用说明。

在 M1 MacBook，macOS 上用 Parallels17 跑 Windows 11 arm 虚拟机，在里面运行 Quartus II，创建项目就会 crash：

Internal Error: Sub-system: DYGR, File: /quartus/ddb/dygr/dygr_place_info.cpp, Line: 3776

在 Intel 的官方论坛这个帖子上找到不少人遇到和我一样的问题，似乎至今没有解决。

目前我迫不得已的解决方案是：用 7 毛钱一个小时的云电脑来运行 Quartus II……😭

221102 Upd：才发现老师指定的版本是 Quartus 9.0（好像是上古版本了……）。在 macOS 虚拟机上运行这个版本没有问题。然而版本从 9 到 14 经历了太多，Altera 公司被 Intel 收购了，Intel 目前官网上不提供老版本的下载。好在我们可以在 Internet Archive 里找到 Quartus II 9.0 Web Edition。

Matlab

本软件白嫖等级为 level1。macOS 平台。

听说是因为 HNU 被美国政府制裁，所以学校不能订购 Matlab 软件。

由于这个软件过于庞大，下载的代价比较大（校园网每个月流量限制 40G 并且速度奇慢）……所以我只好直接使用 MacWk 提供的 R2021b 破解版。这里我暂且信任一下 MacWk 这个网站提供的软件吧……

下载地址

写这篇文章的时候去看了一眼，appstorrent 网站上已经有 R2022a 的破解版下载了（链接）。不过我承担不起这个大小……可以尝试一下。

实在不行，也可以试试 Matlab 的开源替代品：Octave。

LINGO

本软件白嫖等级为 level5。macOS 平台。

没错，LINGO 提供了学生的 License。需要用教育邮箱给 LINDO 公司发一封邮件。

Dear lingo supervisor,

I am a student from Hunan University, China. I major in Information Security and am interested in optimization. In this case, I would like to apply for a full capacity license for further study.

Attached is my user information.

Thank you for reading my message. Looking forward to your reply! 😃

Best Regards,
A user who loves Lingo

在邮件的附件里加上 LINGO 软件里导出的 userinfo 文件，发送邮件之后 48 小时之内 LINDO 公司就会回复，给你一个能用半年多的 License。合法白嫖～

操作系统和虚拟机

Windows 11

本软件白嫖等级为 level4。

记得 Windows 10 刚出的时候，懵懂无知的我马上安装了一个，并用当时似乎很有名的一个叫做 KMS pico 的软件激活了（一键激活的体验确实很好）。后来常常 CPU 莫名接近 100%。几年后了解到，大名鼎鼎的 KMS pico 软件原来植入了挖矿病毒。
经历了这件事情，我从此再也不敢乱装来路不明的破解版了。毕竟让系统变脏很容易，但是变回干净真的很难。（或许恰恰印证了一个系统中熵总是有升高的趋势……）所以老爸问我电脑这么卡怎么办的时候，我一般都建议重装系统。老爸说想用 360 杀一杀毒，我说，别，别……

于是这次又需要激活的时候去问了问 CYP 学长，他找到了这个 Github 项目：Microsoft-Activation-Scripts。尝试了一下，真的很好用！一键就激活了！（而且是完全开源的一套脚本）

只是不知道既然这么容易激活，巨硬还怎么赚钱呢……

Parallels 17

本软件白嫖等级为 level2。macOS 平台。

Parallels Desktop 免费试用期结束之后，并不会删除相应的虚拟机，而是只是在软件内阻止启动。我找到的 PD Runner 软件就是「启动器」，可以直接启动 Parallels 安装的虚拟机，而无视购买要求。目前其 Github repo 地址在这里。

应该说，这个工具并没有「破解」Parallels，而是「绕过」其主程序而启动虚拟机。但是由于其并不开源（Github 只放了一些介绍文件，没有源码），并且要伴随后台运行，所以还是有不安全的可能性。

其他工具

Typora

本软件白嫖等级为 level4。macOS 平台。

找不到第二个颜值能与 Typora 比肩的 Markdown 编辑器了。可惜现在的价格还是有点贵，如果能便宜一点，我是真的愿意支持它的。

Typora 是使用了 web 技术开发的，使用了大量 js，可以直接修改达到「激活」的目的。参考这个教程。

这种「伪激活」的方法有不完美之处，就是会时常弹窗显示「激活成功」。

几个 macOS 盗版软件网站

  • AppsTorrent：俄罗斯有个大名鼎鼎的 TNT 团队，专门破解 macOS 的各种软件，但是这个组织并没有官方的发布渠道。这个网站据说就是汇集了一些来自 TNT 破解的软件。
  • MacWk：国内的盗版软件网站，许多软件也来源于 TNT。有专门的「安装器」。

Upd：本文发布前夕，MacWk 已经关站了。非常可惜。
