我家跨上了“信息高速路”
“太好啦！大好啦！”听妈妈说，我家的电脑跨上了信息高速路，我情不自禁地欢呼起来。电脑网络这个新鲜玩意儿到底是什么呢？
……

以讽 NOI Linux 的复古。9102 年了居然还在 Ubuntu 14.04……

XJ 机房实行严格管理，被迫 Windows7 和 NOILinux 二选一。本来我觉得 NOILinux（32 位的……）并不是设计给人用的，我会毫不犹豫选择了 Win7，但是 Linux 可以没有硬盘保护……所以被迫踏上 NOILinux 这条贼船了……

然而 ubuntu 14.04 被 CCF 改得面目全非，加上 32 位等复古的因素，装什么都有一大堆问题……非人世所堪！

浏览器

个人偏爱 Firefox。然而 noilinux 自带的或者直接 apt 安装都是远古版本……
正确的安装姿势是：先去 firefox 官网下载安装包，解压得到一个 firefox 文件夹。去替换安装目录下的 firefox：

sudo rm -rf /usr/lib/firefox
sudo mv ./firefox /usr/lib/

再启动

编辑器

VSCode

NOILinux 内置的编辑器好像只有 emacs、vim 和 gedit 是能用的……
我偏要 vscode！

vscode 1.36 及以后的版本不再支持 32bit linux 了，所以我们要去下载 1.35 版本……（还好是 May 2019 的，不算太远古）
https://code.visualstudio.com/updates/v1_35

下载完是个 deb 文件，然而 noilinux 阉割了 GUI 的“软件包安装程序”……所以我们要用这个命令手动安装：

sudo dpkg -i code_1.35.1-1560349847_i386

Typora

根据 https://support.typora.io/Typora-on-Linux/ 的指引：

wget -qO - https://typora.io/linux/public-key.asc | sudo apt-key add -

# add Typora's repository
sudo add-apt-repository 'deb https://typora.io/linux ./'
sudo apt-get update

# install typora
sudo apt-get install typora

一些工具

git

直接 sudo apt-get install git 会安装远古版本的……（vscode 表示看不下去了）

所以：

sudo add-apt-repository ppa:git-core/ppa
sudo apt-get update
sudo apt-get install git

exfat filesystem

这个是有些 U 盘的文件系统，ubuntu 默认没有的，插入 exfat 文件系统的 U 盘会挂载失败，显示：

mount: unknown filesystem type 'exfat'

解决方法：

sudo apt-get install exfat-fuse exfat-utils

主题&美化

Unity-tweak-tool

首先当然是 unity-tweak-tool 啦：

sudo apt-get install unity-tweak-tool

直接安装完运行发现弹出窗口：

The following schema is missing

com.canonical.unity.webapps

In order to work properly, Unity Tweak Tool recommends you install the necessary packages

看提示就知道没安装 com.canonical.unity.webapps 这个包。运行以下命令安装：

sudo apt-get install unity-webapps-common

Flatabulous

主题和图标包，这个和正常 ubuntu 无异。（长舒一口气）

sudo add-apt-repository ppa:noobslab/themes
sudo apt-get update
sudo apt-get install flatabulous-theme

sudo add-apt-repository ppa:noobslab/icons
sudo apt-get update
sudo apt-get install ultra-flat-icons

至此，你的 noilinux 能用了！！！
